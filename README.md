# Ai-Resume-Analyser-Recruiter-System

## 🧠 About The Project

The AI Resume Analyzer & Recruiter System is an intelligent application designed to simplify and automate the recruitment process.

In today’s job market, recruiters receive a large number of resumes for a single position, making manual screening time-consuming and inefficient. This project solves that problem by using Artificial Intelligence and Natural Language Processing (NLP) to analyze resumes and match them with job descriptions.

The system extracts important information from resumes, compares it with the required job skills, and calculates a similarity score. Based on this score, candidates are ranked to help recruiters identify the most suitable applicants quickly.

This project also helps job seekers understand how well their resume matches a job role and provides insights for improvement.

---

### ⚙️ How It Works
1. Upload Resume (PDF/Text)  
2. Input Job Description  
3. Text is cleaned and processed  
4. TF-IDF / NLP techniques applied  
5. Similarity score is calculated  
6. Candidates are ranked and displayed  

---

### 🎯 Objective
- Automate resume screening  
- Reduce human effort and bias  
- Improve hiring efficiency  
- Help candidates optimize resumes
